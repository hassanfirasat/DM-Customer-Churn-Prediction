About Us

We are a team of students from IQRA University doing graduation in BSCS department. This website is our Final Year Project for University. We developed a predictive model for churn customers.

  • we have trained our model by using telecom customers data.
  • we have performed Exploratory Data Analysis on data.
  • we have developed Flask API and website.

After building model we developed a Flask API that is responsible for get data from this website than parsing those inputs to the model in model understandable form than the model predicts the probability and returns it to the API than the API sends this data to website were the user sees the prediction.
